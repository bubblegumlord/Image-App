# Aplikacja do przetwarzania obrazów



Aplikacja desktopowa umożliwiająca wykonywanie szerokiego zestawu operacji przetwarzania obrazów monochromatycznych i binarnych, w tym operacji punktowych, filtracji, morfologii oraz analizy obiektów. Projekt został przygotowany w ramach przedmiotu Algorytmy Przetwarzania Obrazów.



### Wymagania



Uruchomienie z kodu źródłowego:

* Python ≥ 3.9



Biblioteki:

* tkinter
* numpy
* Pillow
* opencv-python



### Uruchamianie



1. Otwórz plik 21075\_app.exe w katalogu głównym,
2. Wczytaj obraz z menu Plik → Otwórz,
3. Wybierz operację z menu laboratoriów,
4. Wynik pojawi się w nowej karcie obrazu.
